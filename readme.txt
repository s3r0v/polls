Это простой проект на Django для голосований. Непосредственно функционал для
голосований реализован в приложении polls. Создавать голосования нужно из 
админки. Создавать суперпользователя нужно в консоли, командой:

python manage.py createsuperuser

На опросы можно ответить несколько раз, это не баг, а фича. Сами опросы находятся в *ip*:*port*/polls, на index ничего нет
